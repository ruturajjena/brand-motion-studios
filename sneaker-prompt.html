Sneaker Interactive Website Spec Website Brief · Claude

Instructions for Claude to build an ultra-premium, interactive sneaker launch microsite with interactive floating models, scroll-controlled components, and cinematic flow.

# BR — Built by Ruturaj
## Ultra Premium Interactive Sneaker Experience
Build an ultra-premium sneaker microsite that feels like an Apple keynote combined with Nike's product launch pages. photos are videos are under assets folder. 

The primary goal is NOT ecommerce.
The goal is to create a website that becomes an Instagram viral scroll video.

Everything must feel cinematic.
Every scroll should reveal something new.
No dead sections.
No boring transitions.

---

# DESIGN LANGUAGE

Theme:
• Pure White (#FFFFFF) background
• Premium minimal aesthetic
• Apple + Nothing + Nike design language
• Large whitespace
• Floating objects
• Glassmorphism only where required
• Extremely smooth animations
• 120fps feeling
• Micro interactions everywhere

Typography

Headings:
Huge
Bold
Elegant

Body:
Minimal
Grey
Luxury spacing

Accent:
Black
Very subtle grey
No colorful UI except the sneakers.

---

# WEBSITE FLOW

There are FIVE sections.

Each section should feel completely different.

Every section should transition naturally into the next.

Never simply fade between sections.

Use:
• scale
• depth
• camera movement
• masking
• parallax
• object morphing

throughout the website.

---

# GLOBAL SCROLL SYSTEM

No hard cuts.

Every section morphs naturally.

Camera continuously moves.

Objects transform into next section.

Use

depth

motion

rotation

scale

masking

parallax

to make the website feel alive.

Every 10–15% scroll should reveal something new.

No static moments.

---

# MOUSE INTERACTIONS

Soft magnetic cursor.

Cursor glow.

Subtle distortion around cursor.

Sneakers react naturally.

Cards tilt.

Typography slightly shifts.

Glass reflections move with cursor.

Nothing should feel flat.

# MOTION

Use premium easing.

No linear animations.

Use spring physics.

Organic movement.

Floating physics.

Momentum.

Inertia.

Luxury pacing.

---

# PERFORMANCE

Lazy load videos.

Preload hero assets.

Use GPU accelerated transforms.

Maintain smooth performance.

Avoid animation jank.

# INSTAGRAM VIRAL OPTIMIZATION

The website should create a "wait...what happens next?" feeling.

Every screen recording should contain:

• a reveal
• a transformation
• a satisfying scroll interaction

Avoid empty whitespace without motion.

The first 3 seconds must immediately hook viewers with the floating interactive sneaker.

Every following section should increase visual complexity.

The ending should feel like a product launch trailer rather than an ecommerce website.
