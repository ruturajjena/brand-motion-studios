# What if Apple Launched the Taj Mahal? — Asset Generation Pack
### 8 video clips + 8 still images · generate these separately BEFORE building the website
### Videos go in /assets/videos · Images go in /assets/photos

---

## How to use this pack

1. **Paste the Global Style Block at the END of every prompt.** The generator has no memory between generations — this block keeps the monument, light and color grade consistent so every asset feels like one continuous Apple keynote film.
2. **Generate at the highest resolution available** (4K if possible).
3. **Every video uses ONE slow camera move in ONE direction, constant speed, no cuts.** The website scrubs and blends these clips with scroll — a cut or a reversing camera will feel broken. Regenerate any clip that contains a cut.
4. **No people, no birds close to the lens, no text, no logos, no watermarks** in any asset.
5. Name every export exactly as labeled below — the website build prompt wires assets by filename.

---

## 🎨 GLOBAL STYLE BLOCK (append to every prompt)

> Apple product launch film aesthetic applied to the Taj Mahal: ultra-clean, minimal, museum-grade presentation. Luminous white Makrana marble rendered with physically accurate subsurface glow, soft studio-like natural light, gentle atmospheric haze, enormous negative space in the composition. Cinematic color grade: near-white highlights, warm ivory midtones, soft neutral shadows — calm, premium, restrained, never oversaturated. Shot on cinema camera, 35mm lens, shallow depth of field on macro shots. Photorealistic, 8K architectural photography quality. Extremely slow, perfectly smooth, constant-speed camera motion with zero shake. No people, no text, no logos, no watermarks.

You are an award-winning creative developer and senior frontend engineer.

Your task is to build an incredibly premium, scroll-driven storytelling website called

"What if Apple Launched the Taj Mahal?"
