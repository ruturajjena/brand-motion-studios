# 🎬 Cinematic Luxury Landing Page Prompt: "Golden Hour" by Maison Soléa

## 🎭 The Role
You are an award-winning creative developer who has won Awwwards Site of the Day, FWA Site of the Day, and CSS Design Awards. Your goal is to build a highly viral, immersive cinematic landing page for the fictional luxury fragrance MAISON SOLÉA: Golden Hour Eau de Parfum. 

This page must not feel like a standard e-commerce site; it must feel like an interactive luxury art installation optimized for breathtaking, smooth scroll transitions that make users stop scrolling on their feeds.

---

## 💎 Design Language & Visual Aesthetics
- **Style Archetype**: Apple Event, Saint Laurent, VisionOS, Chanel Beauty, Aesop, and Liquid Glass UI.
- **Color Palette**: 
  - Dark warm background (#030303)
  - Crisp warm ivory text (#F7F5F0)
  - Metallic luxury gold (#D4A843)
  - Frosted translucent charcoal (rgba(12, 12, 12, 0.35))
- **Liquid Glass Panels**: All content containers must look like frosted liquid glass plates:
  - High blur backdrop (backdrop-filter: blur(25px) saturate(1.7))
  - Refractive double-border highlighted edge glares.
  - Subtle gold inner shadow glows (inset 0 0 15px rgba(212,168,67,0.12)).
  - Floating technical "Telemetry" tags in gold (e.g. [02 // ALCHEMICAL FLUID]).
- **Typography**:
  - Serif headers: Cormorant Garamond (light, elegant, low kerning).
  - Sans subtitles and telemetry copy: Outfit or Inter (monospaced vibe, ultra-tracked letter spacing).

---

## 🛠️ Tech Stack & Architecture
- **Framework**: Next.js App Router (TypeScript).
- **Styling**: Pure Vanilla CSS (defined in a single clean globals.css - no Tailwind class bloat).
- **Smooth Scroll**: Lenis Scroll (duration 1.4, custom easing for buttery deceleration).
- **Animations**: GSAP + ScrollTrigger. All element targets must use React refs (avoid string selector query loops to support Strict Mode stability).
- **Background Assets**: Directly overlayed HTML5 <video> background elements (not fragile WebGL textures, to ensure maximum video compatibility, 4K performance, and zero memory leaks).

---

## 🎡 Interactive HUD & Navigation
1. **Frosted Navigation Header**: A floating capsule containing the brand logo and links (Sunset, Essence, Flacon, Alchemy, Mist, Reserve). When clicked, intercept clicks and call programmatic smooth scrolling (lenis.scrollTo).
2. **Sidebar Telemetry HUD**: A vertical progress tracker on the right side of the screen. As sections pin, highlight active links and expand corresponding dots dynamically via ScrollTrigger callbacks.
3. **Ambient Light Grid**: A mouse-following radial gold gradient glow (var(--mouse-x), var(--mouse-y)) applied to the background overlay.
4. **Gold Dust Canvas**: A canvas rendering 50 slow-drifting, semi-transparent golden dust particles rising upward.

---

## 🔒 Custom Liquid Glass Cursor (Desktop)
- Hide the default system pointer (cursor: none).
- Render a customized dual-ring cursor:
  1. Inner Amber Dot: Fast mouse tracking (lerp factor 0.25) with a warm gold glow.
  2. Outer Specular Ring: Lagging inertia (lerp factor 0.12), frosted blur.
- **Dynamic Stretch**: On rapid mouse movement, calculate speed velocity and trajectory angle, stretching the outer ring along its path (max scale 1.35, rotation matching moving direction).
- **Gold Particle Trails**: If movement velocity crosses 3px/frame, spawn tiny gold dust particles that fade out (opacity: 0) and drift randomly over 600ms. Manage the elements in a fixed pool of 25 DOM elements to keep GPU footprint low.
- **Button Hover State**: When hovering over links or call-to-actions, morph the cursor outer ring into a horizontal pill framing the target.

## 🔒 Coding Constraints (Idempotence & Strict Mode)
1. **Prevent Character-Splitting Duplications**: Ensure your text-splitting function is idempotent. Check if .char nodes are already present on the element prior to splitting. If they are, return them directly to prevent rendering raw HTML tags as text during React Strict Mode double-initialization mounts.
2. **Clean Garbage Collection**: Every event listener, GSAP animation ticker, and Lenis scroll listener must be killed and destroyed in the React useEffect cleanup hook return statement to prevent frame drops on pagination cycles.
